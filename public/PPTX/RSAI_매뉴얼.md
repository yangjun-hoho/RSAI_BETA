## 1. 프로그램 소개

RSAI는 남양주시 공무원의 문서 작성·업무 처리를 AI로 자동화하는 내부 업무지원 시스템입니다.

핵심 내용만 입력하면 AI가 초안을 작성하고, 남양주시 공식 서식이 적용된 파일(HWP·PPTX)을 즉시 다운로드할 수 있습니다.

| 구분 | 내용 |
|------|------|
| **주요 기능** | AI 채팅, 공문서 자동 생성, 이미지·영상 제작, 차트 편집 |
| **지원 출력** | HWP(x), PPTX, PNG, MP4 |
| **접속 방법** | 내부망 브라우저에서 접속 (별도 설치 불필요) |
| **이용 대상** | 남양주시 전 직원 |

---

## 2. 기술 스택

| 항목 | 내용 |
|------|------|
| **프레임워크** | Next.js 15 (React) |
| **AI 모델** | OpenAI GPT / Google Gemini (서버 측 API) |
| **문서 생성** | HWPX(ZIP+XML 직접 조작), pptxgenjs |
| **영상 처리** | FFmpeg (숏폼 에디터) |
| **차트** | Chart.js |
| **RAG** | 내부 벡터 검색 엔진 |

## 3. 메인 채팅

화면 중앙의 채팅창에서 자유롭게 AI와 대화할 수 있습니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **웹검색** | 입력창 하단 토글로 ON/OFF. 최신 정보가 필요한 질문에 자동으로 웹 검색 결과를 반영합니다. |
| **모델 선택** | OpenAI(ChatGPT) / Google AI(Gemini) 중 선택. 상황에 따라 전환 가능합니다. |
| **나의 메뉴** | 자주 쓰는 프롬프트·도구를 즐겨찾기처럼 등록해 빠르게 불러옵니다. |
| **대화 초기화** | 상단 버튼으로 대화 내용을 비울 수 있습니다. 이전 대화 맥락이 사라지므로 새 주제 시작 시 사용하세요. |

### 사용 방법

1. 입력창에 질문이나 요청 사항을 입력합니다.
2. `Enter`로 전송, `Shift + Enter`로 줄바꿈합니다.
3. 필요한 경우 웹검색을 ON으로 설정합니다.

> **Tip.** 제목만 입력해도 AI가 내용을 작성해 드립니다. 구체적인 정보(날짜·부서명·수치 등)를 함께 입력할수록 결과 품질이 높아집니다.

## 4. 보고서 생성

업무보고서 초안을 AI가 자동으로 작성합니다.

### 입력 항목

| 항목 | 필수 | 설명 |
|------|------|------|
| **보고서 제목** | ✅ | 제목만 입력해도 생성 가능합니다. |
| **핵심 내용** | 권장 | 포함할 내용·일정·수치 등을 자유롭게 입력합니다. |

### 출력
- 남양주시 업무보고서 서식이 적용된 **HWP 파일** 다운로드
- 생성된 내용을 텍스트로 복사 가능

### 사용 방법

1. 좌측 사이드바에서 **보고서 생성**을 선택합니다.
2. 제목과 핵심 내용을 입력하고 **AI 생성** 버튼을 누릅니다.
3. 미리보기에서 내용을 확인 후 **HWP 다운로드**합니다.
4. 최근 생성 내역은 히스토리에서 다시 불러올 수 있습니다.

## 10. 업무 템플릿

| 템플릿 | 용도 |

## 11. RAG (문서 기반 검색)

업로드된 내부 문서를 기반으로 AI가 질의응답합니다. 조례·지침·매뉴얼 등 내부 자료를 찾을 때 유용합니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **문서 갤러리** | 카테고리별로 등록된 문서 목록을 확인합니다. |
| **RAG 챗봇** | 선택한 카테고리 문서를 기반으로 AI가 답변합니다. |
| **관리자 기능** | 문서 업로드 및 카테고리 관리 (관리자 전용) |

### 사용 방법

1. 좌측 사이드바에서 **RAG**를 선택합니다.
2. 원하는 카테고리(문서 그룹)를 선택합니다.
3. 질문을 입력하면 해당 문서 내용을 기반으로 답변합니다.

> **주의.** 일반 채팅과 달리 업로드된 문서 범위 내에서만 답변합니다. 문서에 없는 내용은 답변하지 않습니다.

## 13. 업무 지원 2

남양주시 공식 서식이 적용된 HWP 파일을 AI가 자동 생성하는 신규 도구 모음입니다. 기존 도구보다 서식 정확도와 완성도가 높습니다.

### 포함 도구

| 도구 | 입력 항목 | 출력 |
|------|----------|------|
| **보고서 생성 v2** | 제목, 핵심 내용 | 업무보고서 HWP |
| **보도자료 생성 v2** | 제목, 핵심 내용 | 보도자료 HWP |
| **인사말씀 생성 v2** | 행사 제목, 내용 | 인사말씀 원고 HWP |
| **공적조서 생성 v2** | 분야, 핵심 공적 | 공적조서 HWP (1장) |
| **확대간부회의 자료 생성** | 템플릿 선택, 제목, 핵심 내용 | 회의자료 HWP (A4 가로 1장) |

### 확대간부회의 자료 — 템플릿 5종

| 템플릿 | 특징 |
|--------|------|
| **기본형** | 이미지·표 없이 텍스트 중심 구성 |
| **교육형** | 교육·연수 안내 및 계획 보고 |
| **사업형** | 사업 추진현황 및 향후계획 보고 |
| **행사1형** | 행사·축제 준비계획 (우측 이미지 삽입 가능) |
| **행사2형** | 기념식·공식 행사 (부제목 포함 레이아웃) |

### 사용 방법

1. 메인 화면 좌측 사이드바에서 **업무 지원 2**를 선택합니다.
2. 원하는 도구를 선택합니다.
3. 제목과 핵심 내용을 입력하고 **AI 생성**합니다.
4. 미리보기 확인 후 **HWP 다운로드**합니다.
5. 최근 생성 내역은 히스토리에서 다시 불러올 수 있습니다.

## 14. Nano Banana 2 (AI 이미지)

텍스트 설명이나 참고 이미지를 기반으로 AI가 이미지를 생성·편집합니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **이미지 생성** | 텍스트 설명(프롬프트)으로 새 이미지 생성 |
| **이미지 편집** | 기존 이미지에 프롬프트로 수정 사항 적용 |
| **마스크 편집** | 이미지의 특정 영역만 선택하여 수정 |

### 입력 항목

| 항목 | 설명 |
|------|------|
| **프롬프트** | 만들고자 하는 이미지를 텍스트로 설명합니다. |
| **참고 이미지** | 편집 시 원본 이미지를 업로드합니다. (최대 10개) |
| **비율** | 1:1, 16:9, 9:16 등 다양한 비율 선택 |
| **해상도** | 512px / 1K / 2K 중 선택 |

### 사용 방법

1. 바로가기 메뉴 또는 환영 화면 카드에서 **Nano Banana 2**를 선택합니다.
2. 생성 / 편집 / 마스크 모드 중 선택합니다.
3. 프롬프트를 입력하고 생성 버튼을 누릅니다.
4. 결과 이미지를 PNG로 다운로드합니다.

> **주의.** 일일 생성 한도: **10회** (전체 사용자 합산 기준)

## 15. 숏폼 에디터

이미지와 텍스트(TTS)를 조합하여 짧은 홍보 영상을 제작합니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **씬 구성** | 이미지 + 나레이션 텍스트로 장면(씬)을 구성합니다. |
| **TTS 음성** | 텍스트를 자동으로 음성으로 변환합니다. |
| **모션 효과** | 줌인·줌아웃·패닝 등 화면 움직임 효과를 적용합니다. |
| **배경음악** | BGM을 선택하여 영상에 삽입합니다. |
| **장면 전환** | 씬 간 자연스러운 전환 효과를 적용합니다. |

### 사용 방법

1. 바로가기 메뉴에서 **숏폼 에디터**를 선택합니다.
2. **씬 추가** 버튼으로 장면을 구성합니다.
   - 이미지를 업로드하고 나레이션 텍스트를 입력합니다.
   - 모션 효과와 전환 방식을 선택합니다.
3. BGM을 선택합니다.
4. **영상 렌더링** 버튼을 누르면 MP4 파일이 생성됩니다.

> **Tip.** 최대 영상 길이는 300초(5분)입니다.

## 16. 차트 에디터

데이터를 입력하면 다양한 형태의 차트를 생성하고 이미지로 저장합니다.

### 지원 차트 유형

`막대형`, `꺾은선형`, `원형(파이)`, `도넛형`, `레이더형`, `극지방형`, `산점도`

### 입력 항목

| 항목 | 설명 |
|------|------|
| **차트 제목** | 차트 상단에 표시되는 제목 |
| **X축 / Y축 레이블** | 각 축의 항목명 (원형 차트는 해당 없음) |
| **데이터** | 항목명과 수치를 행으로 입력합니다. |
| **차트 유형** | 위 7가지 중 선택 |
| **색상 테마** | 파랑·빨강·초록·무지개 등 8가지 |

### 사용 방법

1. 바로가기 메뉴에서 **차트 에디터**를 선택합니다.
2. 차트 제목과 데이터를 입력합니다.
3. 차트 유형과 색상 테마를 선택합니다.
4. 미리보기에서 차트를 확인하고 **이미지로 저장**합니다.

## 18. AI 자유게시판

직원들이 AI 활용 사례, 유용한 프롬프트, 팁을 자유롭게 공유하는 게시판입니다.

### 주요 기능

| 기능 | 설명 |
|------|------|
| **게시글 작성** | AI 활용 사례, 프롬프트 팁, 질문 등을 작성합니다. |
| **댓글** | 다른 직원의 게시글에 의견을 남깁니다. |
| **AI 답변** | 게시글에 AI가 자동으로 참고 답변을 달아줍니다. |

### 사용 방법

1. 좌측 하단 **AI 자유게시판**을 선택합니다.
2. **새 글 작성** 버튼으로 글을 작성합니다.
3. 다른 직원의 게시글을 읽고 댓글로 의견을 공유합니다.

> **Tip.** 업무에 도움이 된 AI 프롬프트나 활용 사례를 공유하면 동료들에게도 큰 도움이 됩니다.
